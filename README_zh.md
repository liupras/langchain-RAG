# langchain+llama3+Chroma RAG demo

#### 介绍
使用langchain+本地大模型+本地矢量数据库搭建RAG系统的演示。

#### 安装教程

1. 安装依赖环境
```cmd
pip install --upgrade langchain langchain-community langchain-chroma bs4
```
